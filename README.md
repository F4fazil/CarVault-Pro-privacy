# CarVault Pro Legal Pages

Static legal site for CarVault Pro.

## Files

- `index.html` - redirects visitors to `/privacy/`
- `terms/index.html` - Terms of Service, served as `/terms/`
- `privacy/index.html` - Privacy Policy, served as `/privacy/`
- `styles.css` - shared app-themed CSS
- `assets/app_icon.png` - app icon used in the header and favicon

## Notes

1. Confirm `support@carvaultpro.com` is the right support/privacy email.
2. Confirm the service provider list matches production.
3. Review the terms and privacy text with a qualified lawyer before launch.

## Deployment

You can deploy this folder as a static site on GitHub Pages, Vercel, Netlify,
or any static host. Use these links inside the app and store listings:

- `/terms/`
- `/privacy/`
